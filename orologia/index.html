Αγγλικά Ελληνικά Περιγραφή
FilesystemΣύστημα ΑρχείωνΤο σύστημα αρχείων είναι η μέθοδος και η δομή που χρησιμοποιείται για την αποθήκευση και οργάνωση αρχείων στον υπολογιστή.
ShellΚέλυφοςΤο κέλυφος είναι το πρόγραμμα που επιτρέπει την αλληλεπίδραση του χρήστη με το λειτουργικό σύστημα μέσω εντολών.
KernelΠυρήναςΟ πυρήνας είναι το βασικό στοιχείο του λειτουργικού συστήματος που διαχειρίζεται τους πόρους του υπολογιστή και επικοινωνεί με το υλικό.
Command Line Interface (CLI)Διεπαφή Γραμμής ΕντολώνΗ διεπαφή γραμμής εντολών είναι ένα εργαλείο που επιτρέπει την αλληλεπίδραση με το λειτουργικό σύστημα μέσω γραμμών κώδικα.
TerminalΤερματικόΤο τερματικό είναι ένα πρόγραμμα που παρέχει πρόσβαση στη γραμμή εντολών του συστήματος.
File SystemΣύστημα ΑρχείωνΤο σύστημα αρχείων είναι η μέθοδος και η δομή που χρησιμοποιείται για την αποθήκευση και οργάνωση αρχείων στον υπολογιστή.
Distribution (Distro)ΔιανομήΗ διανομή είναι μια έκδοση του λειτουργικού συστήματος Linux που περιλαμβάνει τον πυρήνα και διάφορα προγράμματα και εργαλεία.
Package ManagerΔιαχειριστής ΠακέτωνΟ διαχειριστής πακέτων είναι ένα εργαλείο που επιτρέπει την εγκατάσταση, αναβάθμιση και αφαίρεση λογισμικού.
PackageΠακέτοΤο πακέτο είναι ένα σύνολο αρχείων και μεταδεδομένων που περιέχουν λογισμικό και σχετικές εξαρτήσεις, τα οποία μπορούν να εγκατασταθούν ή να διαχειριστούν μέσω ενός διαχειριστή πακέτων.
DaemonΔαίμοναςΟ δαίμονας είναι ένα πρόγραμμα που εκτελείται στο παρασκήνιο και παρέχει διάφορες υπηρεσίες στο λειτουργικό σύστημα.
RootΡίζαΗ ρίζα είναι ο υπέρτατος χρήστης του συστήματος με πλήρη δικαιώματα πρόσβασης σε όλα τα αρχεία και ρυθμίσεις.
PermissionsΔικαιώματαΤα δικαιώματα καθορίζουν ποιοι χρήστες μπορούν να διαβάζουν, να γράφουν ή να εκτελούν ένα αρχείο ή κατάλογο.
ProcessΔιαδικασίαΜια διαδικασία είναι ένα πρόγραμμα που εκτελείται στο σύστημα.
ServiceΥπηρεσίαΗ υπηρεσία είναι ένα πρόγραμμα ή μια διαδικασία που εκτελείται στο παρασκήνιο και προσφέρει συγκεκριμένες λειτουργίες στο σύστημα.
LogΑρχείο ΚαταγραφήςΤο αρχείο καταγραφής είναι ένα αρχείο που αποθηκεύει πληροφορίες σχετικά με τις δραστηριότητες του συστήματος και τα σφάλματα.
Shell ScriptΣενάριο ΚέλυφουςΤο σενάριο κέλυφους είναι ένα αρχείο που περιέχει μια σειρά εντολών που εκτελούνται από το κέλυφος.
Terminal EmulatorΠροσομοιωτής ΤερματικούΟ προσομοιωτής τερματικού είναι ένα πρόγραμμα που μιμείται τη λειτουργία ενός φυσικού τερματικού, επιτρέποντας την αλληλεπίδραση με τη γραμμή εντολών.
Environment VariableΜεταβλητή ΠεριβάλλοντοςΗ μεταβλητή περιβάλλοντος είναι μια μεταβλητή που επηρεάζει τη λειτουργία του λειτουργικού συστήματος και των προγραμμάτων που εκτελούνται στο περιβάλλον του.
Kernel ModuleΜονάδα ΠυρήναΗ μονάδα πυρήνα είναι ένα κομμάτι κώδικα που μπορεί να φορτωθεί και να εκφορτωθεί δυναμικά από τον πυρήνα του λειτουργικού συστήματος για την παροχή επιπλέον λειτουργιών.
Filesystem Hierarchy Standard (FHS)Πρότυπο Ιεραρχίας Συστήματος ΑρχείωνΤο πρότυπο ιεραρχίας συστήματος αρχείων καθορίζει την τυπική δομή και τις θέσεις των αρχείων και καταλόγων σε ένα σύστημα Linux.
Swap SpaceΧώρος ΑνταλλαγήςΟ χώρος ανταλλαγής είναι μια περιοχή του δίσκου που χρησιμοποιείται ως επέκταση της μνήμης RAM όταν η μνήμη είναι γεμάτη.
GRUBGRUBΟ GRUB είναι ο φορτωτής εκκίνησης που χρησιμοποιείται για την επιλογή και φόρτωση του λειτουργικού συστήματος κατά την εκκίνηση του υπολογιστή.
Boot LoaderΦορτωτής ΕκκίνησηςΟ φορτωτής εκκίνησης είναι ένα πρόγραμμα που φορτώνει το λειτουργικό σύστημα στη μνήμη κατά την εκκίνηση του υπολογιστή.
UbuntuUbuntuΤο Ubuntu είναι μια δημοφιλής διανομή του λειτουργικού συστήματος Linux, γνωστή για τη φιλικότητα προς τον χρήστη και την ευκολία εγκατάστασης.
DebianDebianΤο Debian είναι μια διανομή Linux γνωστή για την σταθερότητά της και τη μεγάλη βιβλιοθήκη πακέτων λογισμικού.
Red HatRed HatΗ Red Hat είναι μια εμπορική διανομή Linux που επικεντρώνεται στην υποστήριξη επιχειρήσεων και προσφέρει επαγγελματική υποστήριξη και συντήρηση.
CentOSCentOSΤο CentOS είναι μια δωρεάν και ανοικτού κώδικα διανομή Linux βασισμένη στον κώδικα της Red Hat Enterprise Linux.
Arch LinuxArch LinuxΤο Arch Linux είναι μια διανομή που επικεντρώνεται στη minimalism και την προσαρμογή, προσφέροντας στους χρήστες τη δυνατότητα να χτίσουν το σύστημα σύμφωνα με τις προτιμήσεις τους.
AliasΨευδώνυμοΤο ψευδώνυμο είναι μια συντομευμένη εντολή ή όνομα που αντιπροσωπεύει μια μεγαλύτερη ή πιο σύνθετη εντολή.
Terminal MultiplexerΠολλαπλασιαστής ΤερματικώνΟ πολλαπλασιαστής τερματικών είναι ένα πρόγραμμα που επιτρέπει την αποδοτική διαχείριση πολλών τερματικών συνόδων σε ένα μόνο παράθυρο.
HostnameΌνομα ΥπολογιστήΤο όνομα υπολογιστή είναι το όνομα που δίνεται σε έναν υπολογιστή ή συσκευή για αναγνώριση στο δίκτυο.
SELinux (Security-Enhanced Linux)SELinux (Ασφαλές Linux)Το SELinux είναι μια αρχιτεκτονική ελέγχου πρόσβασης που παρέχει αυξημένη ασφάλεια στο σύστημα με τη χρήση πολιτικών ελέγχου πρόσβασης.
Configuration FileΑρχείο ΡυθμίσεωνΤο αρχείο ρυθμίσεων είναι ένα αρχείο κειμένου που χρησιμοποιείται για την αποθήκευση παραμέτρων και ρυθμίσεων για προγράμματα ή υπηρεσίες.
RepositoryΑποθετήριοΤο αποθετήριο είναι μια συλλογή πακέτων λογισμικού που μπορούν να εγκατασταθούν ή να αναβαθμιστούν μέσω του διαχειριστή πακέτων.
tartarΤο tar είναι ένα εργαλείο για τη δημιουργία και την αποσυμπίεση αρχείων αρχειοθέτησης σε μορφή tar.
