root/
|-- home (Περιέχει τους προσωπικούς φακέλους των χρηστών.)
|-- boot Περιέχει τα αρχεία που απαιτούνται για την εκκίνηση του συστήματος. Αυτό περιλαμβάνει τον πυρήνα του Linux, τα αρχεία παραμετροποίησης του bootloader (όπως το GRUB), τα αρχεία εκκίνησης (initramfs ή initrd) και άλλα αρχεία που αφορούν τη διαδικασία εκκίνησης του συστήματος./
|-- bin  Περιέχει τις βασικές εκτελέσιμες εντολές (εκτελέσιμα προγράμματα) που είναι απαραίτητες για τη λειτουργία του συστήματος και για τη διαχείριση του από τους χρήστες./
|-- dev/
|-- etc/
|   |-- apache2/
|   |    `-- apache2.conf (Αρχείο ρυθμίσεων για τον διακομιστή Apache HTTP.)
|   |-- hosts (Αρχείο που περιέχει τοπικές καταχωρήσεις για τα ονόματα των υπολογιστών και τις αντίστοιχες διευθύνσεις IP.)
|   |-- shadow (Αρχείο που περιέχει κρυπτογραφημένους τους κωδικούς πρόσβασης των χρηστών.)
|   |-- fstab (Αρχείο που καθορίζει τις συσκευές που πρέπει να προσαρτηθούν κατά την εκκίνηση του συστήματος.)
|   |-- sudoers (Αρχείο που καθορίζει τους χρήστες και τις ομάδες που έχουν δικαίωμα να εκτελούν εντολές με δικαιώματα διαχειριστή.)
|   |-- crontab (Αρχείο που περιέχει χρονοπρογραμματισμένες εργασίες (cron jobs).)
|   |-- apt/
|   |    `-- sources.list (Αρχείο που περιέχει τα repositories για το σύστημα διαχείρισης πακέτων APT.)
|   |-- resolv.conf (Αρχείο που καθορίζει τις ρυθμίσεις DNS του συστήματος.)
|   |-- network/
|   |      `-- interfaces (Αρχείο που ορίζει τις ρυθμίσεις δικτύου του συστήματος.)
|   |-- group ( Αρχείο που περιέχει πληροφορίες για τις ομάδες χρηστών.)
|   `-- passwd (Αρχείο που περιέχει πληροφορίες για τους χρήστες του συστήματος.)   
|-- lib (Ο φάκελος lib περιέχει τις βιβλιοθήκες )/ 
|-- mnt/
|-- opt/
|-- proc/
|-- sbin (Ο φάκελος περιέχει εκτελέσιμα αρχεία που χρησιμοποιούνται από το σύστημα για τη διαχείριση και τη συντήρηση του συστήματος.)/
|   |-- reboot (Εκτέλεση επανεκκίνησης του συστήματος.)
|   |-- shutdown (Εκτέλεση απενεργοποίησης του συστήματος.)
|   |-- ifconfig (Ρύθμιση και διαχείριση των δικτυακών διεπαφών.)
|   |-- fsck (Εκτέλεση ελέγχου συστήματος αρχείων και επιδιόρθωση προβλημάτων.)
|   |-- mount (Προσάρτηση αρχείων συστήματος αρχείων σε συγκεκριμένες τοποθεσίες.)
|   |-- dhclient (Δυναμική ανάθεση διευθύνσεων IP μέσω του πρωτοκόλλου DHCP.)
|   |-- iptables (Χρησιμοποιείται για τον έλεγχο του τμήματος φίλτρων πακέτων (firewall) σε πολυάριθμες διανομές Linux.)
|   |-- init (Είναι η πρώτη διεργασία που εκτελείται από τον πυρήνα και είναι υπεύθυνη για την εκκίνηση του συστήματος.)
|   |-- fsck (Εκτελεί έλεγχο και επιδιόρθωση ατελειών στο σύστημα αρχείων.)
|   |-- fdisk (Χρησιμοποιείται για τη διαχείριση των διαμερισμάτων του σκληρού δίσκου.)
|   `-- sysctl (Ρύθμιση και διαχείριση παραμέτρων πυρήνα του συστήματος.)
|-- usr (Ο φάκελος περιέχει τα περισσότερα αρχεία και φακέλους που αφορούν το λειτουργικό σύστημα και τις εφαρμογές που εγκαθίστανται σε αυτό.)/ 
|   |-- bin (Εκτελέσιμα αρχεία που ανήκουν στις εγκατεστημένες εφαρμογές για τους κανονικούς χρήστες.)/
|   |-- lib (Βιβλιοθήκες που χρησιμοποιούνται από τις εφαρμογές.)/
|   |-- include/
|   |-- share/
|   |       `-- man (Εγχειρίδια (man pages) για τις εγκατεστημένες εφαρμογές και εργαλεία.)/
|   |-- local (Εγκατεστημένα προγράμματα και πόροι που προέρχονται από τοπικές εγκαταστάσεις ή προσαρμοσμένες εγκαταστάσεις.)/
|   `-- sbin/
`-- var (Ο φάκελος /var (που προέρχεται από το "variable") περιέχει μεταβαλλόμενα δεδομένα που αλλάζουν κατά τη διάρκεια της λειτουργίας του συστήματος.)/
    |-- lib (Αρχεία βάσεων δεδομένων και άλλα δεδομένα για εφαρμογές, όπως οι βάσεις δεδομένων του συστήματος διαχείρισης βάσεων δεδομένων (database management system).    )
    |-- www (Αρχεία και φάκελοι που αντιστοιχούν στην ιστοσελίδα (web page) του διακομιστή (server) HTTP)
    |-- cache (Αρχεία προσωρινής αποθήκευσης για εφαρμογές, όπως ο προσωρινός φάκελος περιήγησης (browser cache).)
    |-- spool (Αρχεία σειράς αναμονής για εφαρμογές που απαιτούν σειριακή επεξεργασία ή παράδοση δεδομένων, όπως ταχυδρομικά κιβώτια (mailboxes) ή εκτυπωτές.)
    |-- tmp ()
    `-- log (Αρχεία καταγραφής (log files) για διάφορες εφαρμογές και υπηρεσίες του συστήματος.)
