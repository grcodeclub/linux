    Εμφανίζει τα περιεχόμενα ενός καταλόγου.

    ls

    Εμφανίζει τον τρέχοντα κατάλογο εργασίας

    pwd

    Δημιουργεί έναν νέο κατάλογο.

    mkdir

    Αντιγράφει αρχεία και καταλόγους.

    cp

    Μετακινεί ή μεντενένει αρχεία και καταλόγους.

    mv

    Διαγράφει αρχεία ή καταλόγους. (Προσοχή: Μόνο με προσεκτική χρήση!)

    rm

    Βρίσκει αρχεία βάσει διάφορων κριτηρίων.

    find

    Τροποποιεί τα δικαιώματα πρόσβασης σε αρχεία και καταλόγους.

    chmod

    Αλλάζει τον κάτοχο ενός αρχείου ή καταλόγου.

    chown
