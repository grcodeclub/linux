Linux - Ορολογία

Όρος Ερμηνεία
Linux Το λειτουργικό σύστημα ανοιχτού κώδικα βασισμένο στον πυρήνα Linux
Πυρήνας (Kernel) Ο κεντρικός πυρήνας του Linux που διαχειρίζεται τους πόρους του συστήματος
Διανομή (Distribution) Μια εκδοχή του Linux που περιλαμβάνει τον πυρήνα και τα εργαλεία του
Τερματικό (Terminal) Ένα παράθυρο κειμένου για εκτέλεση εντολών
Εντολή (Command) Μια εντολή που εκτελείται από το τερματικό για να διαχειριστεί το σύστημα
Χρήστης (User) Ένα άτομο ή πρόγραμμα με πρόσβαση στο σύστημα
Διαχειριστής (Administrator) Ένας χρήστης με ειδικά δικαιώματα για τη διαχείριση του συστήματος
Προνομιούχος Χρήστης (Root User) Ο χρήστης με τις απόλυτες διαχειριστικές εξουσίες στο σύστημα
Κατάλογος (Directory) Μια δομή που περιέχει αρχεία και άλλους καταλόγους
Διανομή Εκδόσεων (Release Distribution) Μια συγκεκριμένη έκδοση μιας διανομής Linux, όπως Ubuntu 20.04 ή Fedora 33
Διαχειριστής Πακέτων (Package Manager) Ένα εργαλείο που χρησιμοποιείται για τη διαχείριση της εγκατάστασης και της απεγκατάστασης λογισμικού
