Διαχείριση Αρχείων και Καταλόγων (Eντολές)


Αρχεία και Καταλογοι

touch file.txt
cat file.txt
wc -l file.txt

Δημιουργεί έναν νέο κατάλογο.

mkdir

Μετακινεί αρχεία και καταλόγους.

mv

Βρίσκει αρχεία βάσει διάφορων κριτηρίων.

find

Όλες τις χρονοσημάνσεις μαζί με άλλες πληροφορίες για ένα αρχείο εμφανίζονται με την εντολή stat (status).

stat



Πλοήγηση στους καταλόγους

Εμφανίζει τον τρέχοντα κατάλογο εργασίας

pwd

Η μετάβαση σε έναν άλλον κατάλογο του Συστήματος γίνεται με την εντολή cd με όρισμα την απόλυτη ή διαδρομλη του καταλόγου, τον οποίο θέλουμε να κάνουμε τρέχοντα.

cd /home/grcodeclub



Διαγραφή

Διαγράφει αρχεία ή καταλόγους. (Προσοχή: Μόνο με προσεκτική χρήση!)

rm

Το -f (--force) διαγράφει χωρίς ερώτηση

rm -f

Επιβεβαίωση διαγραφής από τον χρήστη 

rm -i

Διαγράφει άδειους καταλόγους

rmdir

Διαγράφει άδειους καταλόγους και εμφανίζει μήνυμα για κάθε κατάλογο που διαγράφεται

rmdir -v

Διαγράφει άδειους καταλόγους και τους γονικούς καταλόγους

rmdir -p

Διαγράφει άδειους καταλόγους

rmdir

Διαγράφει άδειους καταλόγους

rmdir

Διαγράφει άδειους καταλόγους

rmdir

Διαγράφει άδειους καταλόγους

rm -d

Διαγράφει όλα τα αρχεία και τους καταλόγους αναδρομικά

rm -r
rm -R



Περιεχόμενα ενος καταλόγους

Εμφανίζει τον τρέχοντα κατάλογο εργασίας

pwd

Εμφανίζει τα περιεχόμενα ενός καταλόγου.

ls

Το -l ενός επιπρόσθετες πληροφορίες για κάθε αρχείο.

ls -l

Το -a (--all) εμφανίζει και τα αρχεία που το όνομά τους αρχίζει με τελεία (κρυφά αρχεία)

ls -a

ls -t

ls -g

Το -r (--reverse) εμφανίζει τα αρχεία με αντίστροφη σειρά ταξινόμησης

ls -r

ls -i

Το -d (--directory) Εμφανίζει επιπρόσθετες πληροφορίες για κάθε αρχείο

ls -d

Το -R εμφανίζει αναδρομικά τα περιεχόμενα ενός καταλόγου και των υποκαταλόγων του

ls -R

ls -S



Αντιγραφλη

Αντιγράφει αρχεία και καταλόγους.

cp

Για να δημιουργήσουμε ένα αντίγραφο στον ίδιο κατάλογο με το πρωτότυπο αρκεί να δώσουμε αντί για όνομα κατάλογου το νεό όνομα του αρχείου.

cp file.txt newfile.txt



Δικαιώματα

Τροποποιεί τα δικαιώματα πρόσβασης σε αρχεία και καταλόγους.

chmod [file_name]

Execute permission

chmod +x [file_name]

Read permission

chmod +r [file_name]

Write permission

chmod +w

Read, write and execute permissions to the file owner:

chmod u+rwx [file_name]

Read, write and execute permissions to the file group:

chmod g+rwx [file_name]

Read, write and execute permissions to the file All (owner,groups,others):

chmod a+rwx [file_name]

Read, write and execute permissions to the file Others:

chmod o+rwx [file_name]

Remove write permission for the group:

chmod g-w [file_name]

Αλλάζει τον κάτοχο ενός αρχείου ή καταλόγου.

chown
