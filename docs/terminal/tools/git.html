Git-Commands

Αρχικοποιήστε ένα τοπικό αποθετήριο Git

git init

Δημιουργήστε ένα τοπικό αντίγραφο ενός απομακρυσμένου repository

git clone ssh://git@github.com/[username]/[repository-name].git

Ελέγξετε την κατάσταση toy git

git status

Προσθέστε ένα αρχείο

git add [file-name.txt]

Λίστα κλάδων (ο αστερίσκος υποδηλώνει τον τρέχοντα κλάδο)

git branch

Εφαρμογή των αλλαγών στο απομακρυσμένο αποθετήριο

git push origin [branch name]

Ενημέρωση του τοπικού repository

git pull origin [branch name]

Προβολή αλλαγών

git log

Προβολή αλλαγών (Aναλυτικά)

git log --summary

Προβολή αλλαγών (Συνοπτικά)

git log --oneline

Προσθέστε ένα απομακρυσμένο αποθετήριο

git remote add origin ssh://git@github.com/[username]/[repository-name].git

Set a repository's origin branch to SSH

git remote set-url origin ssh://git@github.com/[username]/[repository-name].git

Διαγραφή του branch από το απομακρυσμένο αποθετήριο

git push origin --delete old-branch-name

Εύρεση του Commit ID

git log

Επαναφορά του branch στο συγκεκριμένο commit

git reset --hard [commit-id]

Εκτέλεσε Interactive Rebase

Αντικατάστησε το N με τον αριθμό των commits που θέλεις να εξετάσεις.

git rebase -i HEAD~N

Επιλογή Commit για Διαγραφή: Για να διαγράψεις ένα commit, απλά άλλαξε την λέξη pick σε drop δίπλα από το commit που θέλεις να διαγράψεις.

Δημιουργία νέου branch

git checkout --orphan new-branch-name

Το --orphan δημιουργεί ένα νέο branch χωρίς ιστορικό commits.

Αντικατάσταση του Παλιού Branch

Αν θέλεις να αντικαταστήσεις το παλιό branch με το νέο, μπορείς να διαγράψεις το παλιό branch από το απομακρυσμένο repository και να σπρώξεις το νέο branch:

git push origin --delete old-branch-name
git push origin new-branch-name

Αν θέλεις να σπρώξεις το νέο branch ως το branch που αντικαθιστά το παλιό:

git branch -M old-branch-name
git push -f origin old-branch-name

Παρέχει μια λεπτομερή αναφορά για το μέγεθος και τη δομή του repository, όπως το συνολικό μέγεθος των αντικειμένων, το μέγεθος του ιστορικού και τον αριθμό των αρχείων.
