Terminal

Το Terminal είναι μια εφαρμογή που επιτρέπει στους χρήστες να εκτελούν εντολές και να διαχειρίζονται το σύστημα τους μέσω κειμένου. Μπορεί να χρησιμοποιηθεί για να εκτελέσεις διάφορες λειτουργίες στον υπολογιστή σου, όπως τη δημιουργία, τη μετονομασία, τη μετακίνηση ή τη διαγραφή αρχείων, την εκκίνηση προγραμμάτων, τη διαχείριση των ρυθμίσεων του συστήματος και πολλά άλλα.

Τι πρέπει να προσέχουμε - Tips

  • Tα κεφαλαία και τα πεζά γράμματα παίζουν ρόλο
  • Να προσέχουμε τα κενά
  • Χρησιμοποιούμε το Τab για αυτόματη συμπλήρωση

Πως ανοίγω το τερματικό

  1. Alt+Ctrl+T
  2. Alt+F2 και πληκτρολογούμε “terminal”
  3. Applications >> Accessories >> Terminal
