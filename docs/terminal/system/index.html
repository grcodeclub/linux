Διαχείριση συστήματος

Εμφανίζει πληροφορίες για το σύστημα, όπως το όνομα του πυρήνα, την έκδοση κ.ά.

uname -a
-a, --all print all information, in the following order
-s, --kernel-nameprint the kernel name
-n, --nodenameprint the network node hostname
-r, --kernel-releaseprint the kernel release
-m, --machineprint the machine hardware name
-v, --kernel-versionprint the kernel version
-p, --processorprint the processor type (non-portable)
-i, --hardware-platformprint the hardware platform (non-portable)
-o, --operating-systemprint the operating system

Εμφανίζει πληροφορίες σχετικά με την έκδοση της διανομής.

lsb_release -a

Διεργασίες

Η εντολή top είναι μια άλλη εντολή που χρησιμοποιείται στο Linux για την παρακολούθηση των τρεχουσών διεργασιών στο σύστημα. Η top εμφανίζει μια δυναμικά ενημερωμένη λίστα με τις διεργασίες που τρέχουν στο σύστημα, με πληροφορίες όπως το PID (Process ID), το % CPU χρήσης, το % μνήμης, και άλλες σχετικές πληροφορίες. Μπορείτε επίσης να χρησιμοποιήσετε την top για να ταξινομήσετε τις διεργασίες ανά CPU, μνήμη, κλπ., και για να διαχειριστείτε τις διεργασίες από τη γραμμή εντολών.

top

Το htop είναι μια βελτιωμένη έκδοση της εντολής top στο Linux. 

htop

Η εντολή top εμφανίζει πληροφορίες σχετικά με τις τρέχουσες διεργασίες (processes) στο σύστημα.

ps

du -h

fdisk

gdisk
gparted
