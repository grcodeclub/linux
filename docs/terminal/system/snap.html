Snap

Εγκατάτασταση ενός πακέτου με snap

sudo snap install [όνομα-πακέτου]

Αφαίρεση ενός πακέτου με snap

sudo snap remove [όνομα-πακέτου]

Λίστα με όλα τα εγκατεστημένα snap πακέτα

snap list

Αναβάθμιση ενός εγκαταστημένου πακέτου snap

snpa refresh [όνομα-πακέτου]

Επαναφορά ενός πακέτου σε προηγούμενη έκδοση

snap revert [όνομα-πακέτου]

Εμφάνιση πληροφοριών για ένα snap πακέτο

snap infpo [όνομα-πακέτου]

Εγκατάτασταση του Snapd 

sudo apt install snand

Έλεγχος της Εγκατάταστασης του Snap

snap --version

Ενεργοποίση του Snapd

sudo systemctl enable --now snapd.socket

Αναζήτηση του προβλήματος στο Snap

snap warings
