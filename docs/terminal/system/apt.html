Apt

Ανανεώνει τη λίστα των διαθέσιμων πακέτων και των εκδόσεών τους.

sudo apt update

Αναβαθμίζει όλα τα εγκατεστημένα πακέτα στο σύστημα στις τελευταίες διαθέσιμες εκδόσεις.

sudo apt upgrade

Όπως το apt upgrade, αλλά με την προσθήκη ότι αφαιρεί πακέτα αν αυτό είναι αναγκαίο για την αναβάθμιση άλλων.

sudo apt full-upgrade

Αφαιρεί το πακέτο με το όνομα [package_name] χωρίς να διαγράφει τα αρχεία ρυθμίσεων.

sudo apt remove [package_name]

Αφαιρεί πακέτα που εγκαταστάθηκαν ως εξαρτήσεις και δεν χρειάζονται πλέον.

sudo apt autoremove

Εμφανίζει μια λίστα πακέτων. Μπορεί να χρησιμοποιηθεί με διάφορες επιλογές.

sudo apt list

Αναζητά πακέτα που ταιριάζουν με τον όρο αναζήτησης [search_term]

sudo apt search [search_term]

Εμφανίζει λεπτομερείς πληροφορίες για το πακέτο με το όνομα [package_name].

sudo apt show [package_name]

Καθαρίζει την κρυφή μνήμη από τα ληφθέντα πακέτα.

sudo apt clean

Καθαρίζει την κρυφή μνήμη από πακέτα που δεν είναι πλέον διαθέσιμα στο αποθετήριο.

sudo apt autoclean
