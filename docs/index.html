Λειτουργικό σύστημα Linux

Το Linux είναι ένα λειτουργικό σύστημα όπως είναι και τα Winodws. Σε σχέση με τα υπόλοιπα λειτουργικά συστήματα, τα Linux είναι λειτουργικό σύστημα ανοιχτού κώδικα και παρέχονται δωρεάν. Είναι παρόμοια με τα UNIX αλλά όλος ο πηγαίος κώδικας έχει γραφτεί από την αρχή ως ελεύθερο λογισμικό υπό την ελεύθερη άδεια χρήσης GNU General Public License.
