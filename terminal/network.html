Εντολές Δικτύου

netstat

Εμφανίζει πληροφορίες για τις δικτυακές συνδέσεις, τις πόρτες που ακούν και άλλα στατιστικά δικτύου.

netstat

List all TCP ports

netstat -at

List all UDP ports

netstat -au

Για να επιστρέψετε μια λίστα μόνο με τα listening για όλα τα πρωτόκολλα, χρησιμοποιήστε:

netstat -l

Για να επιστρέψετε μια λίστα μόνο με τα TCP ports που είναι listening, χρησιμοποιήστε:

netstat -lt

Για να επιστρέψετε μια λίστα μόνο με τα UDP ports που είναι listening, χρησιμοποιήστε:

netstat -lu

Εμφάνιση στατιστικών στοιχείων για όλες τις θύρες ανεξάρτητα από το πρωτόκολλο

netstat -s

Εμφάνιση στατιστικών στοιχείων μόνο για TCP ports

netstat -st

ss

Είναι αντικατάσταση του netstat.

ss

ping

Αποστέλλει μια ICMP αίτηση προς έναν καθορισμένο προορισμό για να ελέγξει την συνδεσιμότητα και τον χρόνο απόκρισης.

ping www.google.com

traceroute

Εμφανίζει τη διαδρομή που ακολουθείται από ένα πακέτο κατά τη μετάδοσή του από την αρχική πηγή προς τον προορισμό.

traceroute  www.google.com

nslookup

Εκτελεί αντίστροφη αναζήτηση DNS για να ανακτήσει πληροφορίες σχετικά με έναν καθορισμένο κεντρικό διακομιστή ονομάτων.

ssh

Επιτρέπει ασφαλή απομακρυσμένη σύνδεση με έναν εξυπηρετητή μέσω του πρωτοκόλλου SSH.

ssh

tcpdump

Η εντολή tcpdump έχει σχεδιαστεί για τη λήψη και την εμφάνιση πακέτων.

Πριν ξεκινήσετε οποιαδήποτε λήψη, πρέπει να γνωρίζετε ποιες διεπαφές μπορεί να χρησιμοποιήσει το tcpdump.

tcpdump -D
1.wlp3s0 [Up, Running, Wireless, Associated]
2.any (Pseudo-device that captures on all interfaces) [Up, Running]
3.lo [Up, Running, Loopback]
4.enp0s25 [Up, Disconnected]
5.docker0 [Up, Disconnected]
6.bluetooth-monitor (Bluetooth Linux Monitor) [Wireless]
7.nflog (Linux netfilter log (NFLOG) interface) [none]
8.nfqueue (Linux netfilter queue (NFQUEUE) interface) [none]
9.dbus-system (D-Bus system bus) [none]
10.dbus-session (D-Bus session bus) [none]

Εάν θέλετε να καταγράψετε το traffic στο wlp3s0.

tcpdump -i wlp3s0

Μπορείτε να φιλτράρετε το traffic που προέρχεται από έναν συγκεκριμένο host.
Για παράδειγμα, για να βρείτε το trafic που προέρχεται ή πηγαίνει στο 8.8.8.8, χρησιμοποιήστε την εντολή:

sudo tcpdump -i wlp3s0 host 8.8.8.8

Για το traffic που έχει ως αποστολέα το 8.8.8.8, χρησιμοποιήστε:

tcpdump -i wlp3s0 src host 8.8.8.8

Για το traffic έχει ως παραλήπτη το 8.8.8.8, χρησιμοποιήστε:

tcpdump -i wlp3s0 dst host 8.8.8.8

Καταγράψτε την κίνηση από και προς ένα δίκτυο

tcpdump -i wlp3s0 net 192.168.1.0 mask 255.255.255.0

ή

tcpdump -i wlp3s0 net 192.168.1.0/24

Με βάση τον αποστολέα

sudo tcpdump -i wlp3s0 src net 192.168.1.0/24

Με βάση τον προορισμό

sudo tcpdump -i wlp3s0 dst net 192.168.1.0/24

Καταγράψτε την κίνηση από και προς σε ένα port number

DNS port 53

tcpdump -i wlp3s0 port 53

Δημόσια Διεύθυνση IP

Η συγκεκριμένη εντολή χρησιμοποιείται για να λάβεις τη δημόσια Διεύθυνση IP του δικτύου από τον ιστότοπο "ifconfig.me"

curl -s ifconfig.me
